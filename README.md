# 지수 구간 계산

KOSPI와 NASDAQ의 전저점·전고점을 기준으로 현재 지수 위치를 상승/하락 온도계 형태로 시각화하는 웹앱입니다.

환율, 달러지수, 미국 장단기 금리차, CNN 공포탐욕지수 같은 매크로 지표도 함께 확인할 수 있습니다.

## 바로 사용하기

- 배포 사이트: https://index-temperature-calculator.vercel.app/
- GitHub 저장소: https://github.com/ninilee3050/index-temperature-calculator

이 저장소의 파일은 두 가지 방식으로 사용할 수 있습니다.

- 로컬 앱: `지수 구간 계산 앱.cmd`를 더블클릭해서 내 컴퓨터에서 실행
- 웹사이트: GitHub `main` 브랜치에 올리면 Vercel이 자동 배포

## 주요 기능

- KOSPI / NASDAQ 지수 온도계
- KOSPI / NASDAQ 현재 지수 포인트 자동 갱신
- 전저점 / 전고점 기준 수동 설정
- 상승 구간 / 하락 구간 동시 시각화
- 현재 수은과 최고/최저 도달 수은 이중 표시
- 최고/최저 도달 퍼센트와 마우스오버 기록일 표시
- 달러 환율, 달러지수, 미국 장단기 금리차 표시
- CNN 공포탐욕지수 표시
- 라이트 / 다크 모드 지원

## 데이터 업데이트 방식

`새로고침` 버튼을 누르면 매크로 지표를 다시 확인합니다.

다만 외부 데이터 요청이 너무 잦아지는 것을 막기 위해 서버는 약 5분 동안 최근 데이터를 캐시합니다. 그래서 새로고침 버튼을 눌러도 5분 이내에는 `UPDATED` 시간이 바로 바뀌지 않을 수 있습니다.

- `UPDATED` 시간이 바뀜: 서버가 새 데이터를 다시 받아온 상태
- `UPDATED` 시간이 그대로임: 새로고침은 했지만 아직 캐시된 최신 데이터를 사용 중인 상태

현재 지수 온도계의 수은 위치는 KOSPI / NASDAQ 현재 지수 포인트를 기준으로 계산됩니다. 기준 설정에서 전저점·전고점 값을 바꾸면 구간별 지수 포인트도 다시 계산됩니다.

## 데이터 출처

- KOSPI / NASDAQ: Yahoo Finance
- 달러 실시간 환율: Yahoo Finance
- 달러 지수: Yahoo Finance, 필요 시 Stooq 보조 사용
- 미국 장단기 금리차 10Y-2Y, 10Y-3M: Yahoo Finance 금리 데이터를 이용해 직접 계산
- CNN 공포탐욕지수: CNN Fear & Greed
- 장단기 금리차 양전 구간 분석: FRED 데이터 사용

## 인터넷 사이트로 사용하기

이 저장소는 Vercel에 배포할 수 있습니다. 현재 배포 주소는 아래와 같습니다.

```text
https://index-temperature-calculator.vercel.app/
```

Vercel에 GitHub 저장소를 연결하면 아래 흐름으로 동작합니다.

```text
내 컴퓨터에서 수정
→ GitHub main 브랜치에 업로드
→ Vercel이 자동 배포
→ https://...vercel.app 주소로 접속
```

### Vercel 배포 방법

1. Vercel에 로그인합니다.
2. `Add New...` 또는 `New Project`를 누릅니다.
3. GitHub 저장소 `ninilee3050/index-temperature-calculator`를 선택합니다.
4. Framework Preset은 `Other` 또는 자동 감지 값을 그대로 둡니다.
5. Build Command는 비워둬도 됩니다.
6. Output Directory도 비워둬도 됩니다.
7. Deploy를 누릅니다.
8. 배포가 끝나면 `https://...vercel.app` 주소가 생성됩니다.

이후 `main` 브랜치에 새 커밋을 push하면 Vercel이 새 버전을 자동으로 배포합니다.

## 로컬과 Vercel의 차이

같은 파일을 사용하지만 실행 위치만 다릅니다.

- 로컬 실행: 내 컴퓨터의 Node.js 서버가 데이터를 가져옵니다.
- Vercel 실행: Vercel의 서버 함수가 데이터를 가져옵니다.

앱 화면, 계산 로직, 디자인은 같은 파일을 사용합니다. 따라서 GitHub에 있는 파일은 로컬에서도 실행되고 Vercel에서도 작동합니다.

## 무료 사용 참고

이 앱은 개인용 또는 소규모 공유용으로 가볍게 사용할 수 있는 구조입니다.

- 데이터베이스를 사용하지 않습니다.
- 로그인 기능을 사용하지 않습니다.
- 결제 기능이나 개인정보 입력 기능이 없습니다.
- 서버는 `/api/macro`, `/api/yield-cycles` 데이터 조회만 담당합니다.
- 방문자가 많아지면 Vercel 무료 한도나 외부 데이터 소스 제한에 걸릴 수 있습니다.
- 외부 공개 데이터 소스가 일시적으로 막히면 일부 지표가 표시되지 않을 수 있습니다.

URL을 아는 사람은 접속할 수 있으므로 완전한 비공개 앱은 아닙니다. 혼자 쓰는 용도라면 주소를 공유하지 않는 방식으로 충분히 가볍게 운영할 수 있습니다.

## 로컬 실행 조건

- Windows
- Node.js 18 이상
- 인터넷 연결
- Chrome, Edge 등 최신 브라우저

Node.js가 설치되어 있지 않으면 `지수 구간 계산 앱.cmd` 실행 시 안내창이 표시됩니다.

## 로컬 실행 방법

1. 이 저장소를 다운로드합니다.
2. Node.js 18 이상을 설치합니다.
3. `지수 구간 계산 앱.cmd`를 더블클릭합니다.
4. 브라우저에서 앱이 자동으로 열립니다.

개발용으로 직접 실행할 때는 아래 명령을 사용할 수 있습니다.

```bash
npm start
```

## 사용 방법

- `기준 설정` 버튼에서 각 시장의 전저점 날짜/포인트와 전고점 날짜/포인트를 입력합니다.
- 입력값이 바뀌면 지수 온도계의 구간별 지수 포인트가 자동으로 다시 계산됩니다.
- `새로고침` 버튼을 누르면 매크로 지표를 다시 불러옵니다.
- 새로고침 후 `UPDATED` 시간이 바로 바뀌지 않으면 약 5분 캐시가 유지 중인 상태입니다.
- 상단 전등 버튼으로 라이트/다크 모드를 전환할 수 있습니다.
- 설정값과 테마는 브라우저별로 저장됩니다.

## 파일 구성

```text
index.html              앱 화면
app.js                  앱 계산 로직과 화면 동작
styles.css              UI 스타일
server.mjs              로컬 앱 서버
lib/market-data.mjs     실시간 데이터 수집 공용 로직
api/macro.js            Vercel 매크로 데이터 API
api/yield-cycles.js     Vercel 장단기 금리차 양전 구간 API
vercel.json             Vercel 배포 설정
지수 구간 계산 앱.cmd    로컬 실행 파일
scripts/launch-app.ps1  로컬 실행 보조 스크립트
```
