# 지수 구간 계산

KOSPI와 NASDAQ의 전저점·전고점을 기준으로 현재 지수 위치를 상승/하락 온도계 형태로 시각화하는 로컬 웹앱입니다.

환율, 달러지수, 미국 장단기 금리차, CNN 공포탐욕지수 같은 매크로 지표도 함께 확인할 수 있습니다.

## 주요 기능

- KOSPI / NASDAQ 지수 온도계
- KOSPI / NASDAQ 현재 지수 포인트 자동 갱신
- 전저점 / 전고점 기준 수동 설정
- 상승 구간 / 하락 구간 동시 시각화
- 달러 환율, 달러지수, 미국 장단기 금리차 표시
- CNN 공포탐욕지수 표시
- 라이트 / 다크 모드 지원
- 브라우저를 닫으면 로컬 서버가 자동 종료되는 실행 방식

## 실행 조건

- Windows
- Node.js 18 이상
- 인터넷 연결
- Chrome, Edge 등 최신 브라우저

Node.js가 설치되어 있지 않으면 `지수 구간 계산 앱.cmd` 실행 시 안내창이 표시됩니다.

## 실행 방법

1. 이 저장소를 다운로드합니다.
2. Node.js 18 이상을 설치합니다.
3. `지수 구간 계산 앱.cmd`를 더블클릭합니다.
4. 브라우저에서 앱이 자동으로 열립니다.

## 사용 방법

- `기준 설정` 버튼에서 각 시장의 전저점 날짜/포인트와 전고점 날짜/포인트를 입력합니다.
- 입력값이 바뀌면 지수 온도계의 구간별 지수 포인트가 자동으로 다시 계산됩니다.
- `새로고침` 버튼을 누르면 매크로 지표를 다시 불러옵니다.
- 상단 전등 버튼으로 라이트/다크 모드를 전환할 수 있습니다.

## 파일 구성

```text
index.html              앱 화면
app.js                  앱 계산 로직과 화면 동작
styles.css              UI 스타일
server.mjs              로컬 앱 서버와 실시간 데이터 연동
지수 구간 계산 앱.cmd    실행 파일
scripts/launch-app.ps1  실행 보조 스크립트
```

## 참고

이 앱은 로컬 컴퓨터에서 실행되는 웹앱입니다.  
실시간 데이터는 외부 공개 데이터 소스 상태에 따라 일시적으로 불러오지 못할 수 있습니다.
